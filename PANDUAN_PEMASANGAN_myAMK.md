# PANDUAN PEMASANGAN myAMK

## KAEDAH 1: FAIL TEMPATAN (TANPA INTERNET SELEPAS MUAT TURUN)

### 💻 Desktop/Laptop:
1. Muat turun fail `myAMK_App.html`
2. Simpan di folder yang mudah dicari (cth: Desktop)
3. Double-click fail → buka di Chrome/Edge/Firefox
4. Bookmark halaman untuk akses mudah

### 📱 Phone (Android):
1. Muat turun fail `myAMK_App.html`
2. Buka Chrome → Pergi ke `chrome://flags` → Cari "file" → Enable
3. Atau gunakan app "File Manager" → buka fail dengan Chrome
4. Buka menu ⋮ → "Add to Home screen" → Namakan "myAMK"
5. Icon myAMK muncul di home screen!

### 📱 Phone (iPhone):
1. Muat turun fail `myAMK_App.html`
2. Buka di Safari melalui Files app
3. Tekan ikon Share □↑ → "Add to Home Screen"
4. Icon myAMK muncul di home screen!

## KAEDAH 2: GITHUB PAGES (PERCUMA, BOLEH AKSES DARI MANA-MANA)

### Langkah 1: Buat Akaun GitHub (jika belum ada)
1. Pergi ke https://github.com
2. Klik "Sign up"
3. Isi email, password, username
4. Verify email

### Langkah 2: Buat Repository Baru
1. Login ke GitHub
2. Klik butang "+" di kanan atas → "New repository"
3. Repository name: `myamk`
4. Pilih "Public"
5. ✅ Tick "Add a README file"
6. Klik "Create repository"

### Langkah 3: Upload Fail
1. Dalam repository, klik "Add file" → "Upload files"
2. Drag & drop fail `myAMK_App.html`
3. **PENTING**: Rename fail kepada `index.html` sebelum upload
   - Atau selepas upload, klik fail → Edit → tukar nama ke `index.html`
4. Klik "Commit changes"

### Langkah 4: Aktifkan GitHub Pages
1. Pergi ke tab "Settings" dalam repository
2. Di sidebar kiri, klik "Pages"
3. Bawah "Source", pilih "Deploy from a branch"
4. Branch: pilih "main", folder: "/ (root)"
5. Klik "Save"
6. Tunggu 1-2 minit

### Langkah 5: Akses App Anda!
1. URL app anda: `https://[username-anda].github.io/myamk/`
2. Contoh: `https://ahmad123.github.io/myamk/`

### Langkah 6: Install di Phone sebagai App
**Android:**
1. Buka URL di Chrome
2. Chrome akan tunjuk banner "Add myAMK to Home screen" → Tekan "Install"
3. Atau tekan ⋮ → "Install app" atau "Add to Home screen"
4. myAMK muncul sebagai app sebenar di phone!

**iPhone:**
1. Buka URL di Safari
2. Tekan ikon Share □↑
3. Pilih "Add to Home Screen"
4. myAMK muncul sebagai app di phone!
